# Java DBMS — Аналог PostgreSQL

**Java DBMS** — это учебный проект, реализующий базовую функциональность системы управления базами данных (СУБД), похожей на PostgreSQL, полностью на Java. Проект предназначен для изучения внутренней архитектуры СУБД, включая обработку SQL-запросов, планирование, оптимизацию, индексы и работу с транзакциями.

---

## 🏗️ Функциональность

- Парсинг и выполнение базовых SQL-запросов (`SELECT`, `INSERT`, `UPDATE`, `DELETE`, `CREATE TABLE`, `CREATE INDEX`)
- Планирование выполнения запросов
- Оптимизация физических планов
- Поддержка индексов для ускорения поиска
- Работа с транзакциями и простая имитация WAL (Write-Ahead Log)
- Модульное хранение данных (каталоги таблиц, индексов, типов)
- Поддержка работы с несколькими типами данных
